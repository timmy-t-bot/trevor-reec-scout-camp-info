## About This Repository

This repository contains important information for parents with scouts attending the Trevor Reec Scout Camp, specifically for Week 2. The information includes:

1. **Packing List** - A comprehensive checklist of items scouts should bring to camp
2. **Camp Schedule** - Detailed daily schedule with times and activities for Week 2
3. **Merit Badges** - Information about merit badges available during Week 2
4. **Important Links** - Useful resources and external websites for parents

## Important Note

Specific information about Trevor Reec camp could not be found through research. The information provided here is based on typical Boy Scout camp programming for advanced weeks and should be verified with your troop leadership or camp administration for accuracy.

## How to Use This Repository

If you're viewing this on GitHub:
1. Click on any of the HTML files to view them directly in your browser
2. Download the entire repository as a ZIP file (Code > Download ZIP)
3. Clone the repository to your computer for offline access

## Publishing as a Website

This repository is configured to work with GitHub Pages:
1. Go to Settings > Pages
2. Select "Deploy from a branch"
3. Choose the main branch
4. Your website will be available at https://[username].github.io/trevor-reec-scout-camp/
